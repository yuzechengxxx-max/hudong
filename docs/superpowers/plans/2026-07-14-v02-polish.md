# FlowFilm v0.2 Polish Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Finish FlowFilm v0.2 with reliable additive marquee selection, complete media thumbnails, visible save and control feedback, a persistent minimap toggle, basic project settings, and a verified 300-node editing fixture.

**Architecture:** React Flow remains the owner of transient graph interaction state while project changes are committed at transaction boundaries. Project settings are schema-backed project data, minimap visibility is editor-local state, and manual/automatic saves share one storage helper and one status model. Visual polish is implemented through CSS theme tokens rather than per-button overrides.

**Tech Stack:** React 19, TypeScript 5.8, Vite 7, Vitest, Testing Library, React Flow 12, Zod 3, Lucide React.

```ts
export function createLargeProject(count: number): Project {
  const base = createStarterProject();
  const nodes = Array.from({ length: count }, (_, index): StoryNode => ({
    id: index === 0 ? "start" : `scene-${index}`,
    kind: index === 0 ? "start" : "scene",
    title: index === 0 ? "故事入口" : `场景 ${index}`,
    position: { x: (index % 20) * 240, y: Math.floor(index / 20) * 170 },
    ...(index === 0 ? {} : { mediaUrl: "", speaker: "", dialogue: "", showDialogue: false }),
  } as StoryNode));
  return { ...base, nodes, edges: nodes.slice(1).map((node, index) => ({
    id: `edge-${index}`,
    source: nodes[index].id,
    sourcePort: "next",
    target: node.id,
  })) };
}
```
